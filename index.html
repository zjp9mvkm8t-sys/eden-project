SCENARIOS

3개 시나리오

각각 다른 장르와 진실을 담고 있습니다. 모든 시나리오는 에덴의 비밀을 다른 시각에서 조명합니다.

SCENARIO · A
에덴의 가면
심리 스릴러 × 바이오호러
절친 규리를 찾아 위장 전입한 서연. 완벽한 모범생 수인을 믿고 따르지만, 믿었던 손길이 가장 위험한 함정이었다. 2겹 반전 구조로 불일치 해소 원리를 완전 구현.
QUESTS 11
엔딩 3종
SCENARIO · B
무균실의 기억
SF 디스토피아 × 탈출 미스터리
규리가 미리 숨겨둔 암호와 기록이 탈출의 전부. 먼저 탈출한 아란의 기억마저 조작됐다면? QR·모스부호·노트 교차 분석의 SF적 퍼즐 구조.
QUESTS 7
엔딩 2종
SCENARIO · C
선택받은 자들
도덕 공포 × 사회 비판 스릴러
수인은 에덴의 철학을 진심으로 믿는다. 탈출 후에도 선택해야 한다 — 선별 기준표를 세상에 공개할 것인가. 퍼즐보다 도덕적 선택이 핵심.

CHAPTER 12 APPLIED

퍼즐 유형 설계

Chapter 12 방탈출 기본 문제 유형 7가지를 에덴 스토리에 맞게 재해석했습니다.

🔍
TYPE 01
수색 / 관찰
모든 스테이지
소품 클릭 인터랙션. 액자 뒤 팔찌, 인형 배 속 쪽지, 좌석 하단 포스트잇. 환경 배치 원칙 적용.
🔢
TYPE 02
개수 세기
A-04, C-02
학생회장 사진 재임 연도 세기 → 금고 비번. X 표시된 학생 번호 순서 나열 → 금고 개방.
🧬
TYPE 03
패턴 규칙
A-03, B-01
규리 노트의 해부학 기호 → 숫자 치환표. 약품 라벨과 교차 분석 시 B-3-47 확정.
🔐
TYPE 04
카이사르 암호
A-06, B-03
이동 칸수 4 확정. EDEN → IHIR. 규리 노트 암호문을 4칸 역이동하면 탈출 경로 해독.
🎨
TYPE 05
색상 치환
A-06
약품 라벨 색상 코드 매핑. 색상 순서가 숫자로 치환되어 냉장고 자물쇠 번호 구성.
🧠
TYPE 06
논리 추론
A-09, C-05
수인 vs 아란 발화. 셋 중 하나만 진실 — 수인 발화 2개 중 하나는 반드시 거짓. 아란 루트가 정답.
🧩
TYPE 07
그림 퍼즐
A-08
해부 인형 복부 지퍼 인터랙션. 규리가 숨겨둔 쪽지 조합 → 수인 배지 = 마스터키 확정.

CHAPTER 10 APPLIED

ARG Rabbit Hole 설계

TINAG 원칙 기반 5개 진입 장치. 클라우드 메이커가 6개 단서를 수집하면 방탈출 예약 코드를 받습니다.

RH-01
ATTENTION
SNS · 트위터 / 인스타그램 / 에브리타임
규리 실종 제보 게시글
CLUE-01: "배지 달린 애 조심해. 해부 인형 배 속 봐."
@규리를_찾아주세요 계정. 퍼펫 마스터가 댓글 실시간 응답하며 현실감 유지.
RH-02
SEARCH
HTML · 공식 홈페이지 소스코드 주석
개발자 도구(F12)에 숨겨진 단서
CLUE-02: PID_0047 / LAB_B3 / BADGE_KEY=MASTER
IT 능숙한 클라우드 메이커가 발견 → 커뮤니티에 공유. ?ref=find_me 파라미터로 규리 비밀 메모 페이지 진입.
RH-03
SEARCH
YouTube · 학원 공식 채널 홍보 영상
14:88 타임스탬프 배경음 모스부호
CLUE-03: SOS + HELP + 0047
비정상 타임스탬프(14:88)에 모스부호 삽입. 모스부호 분석기로 해독 시 규리 실험체 번호 0047 확정.
RH-04
ACTION
오프라인 · 지하철역·카페 전단지 UV 잉크
UV 조명 아래서만 보이는 메시지
CLUE-04: "거칠어 보이는 자를 믿어라."
QR 스캔 시 정상 홈페이지. ?ref=find_me 포함된 UV QR만 비밀 페이지 진입. 아란이 구원자임을 암시.
RH-05
SHARE
에브리타임 · 대학 익명 커뮤니티
재학생 위장 익명 제보글 + 퍼펫마스터 DM
CLUE-05: 사물함 비번 = 생일 역순 → 5130
댓글 힌트 요청 시 DM 발송. SQ-1 사물함 자물쇠 정답을 ARG 단계에서 미리 제공.

DECISION LOG

주요 확정 결정사항

D-001부터 D-014까지 프로젝트 전반의 핵심 결정 사항입니다.

D-001
시나리오 3개 확정 — A(심리스릴러) · B(SF디스토피아) · C(도덕공포). Agentic Workflow 4인팀(지헌·지원·나경·하영) 협업 결과.
D-002
캐릭터 외형 확정 — 교복: 네이비 블레이저 + 체크 넥타이. 서연(긴 직모/청순) · 수인(비녀/차가운) · 아란(칼단발/센) · 규리(귀여운/창백).
D-003
퀘스트 구성표 v3 완성 — 4개 시트. 24개 퀘스트 × 13개 컬럼(NPC 스크립트·BGM·카메라·난이도·힌트·타이머 포함).
D-004
ARG Rabbit Hole 설계 완성 — 5개 진입 장치. 6개 CLUE. CLUE-06 규리 노트가 1막 획득 → 3막 최종 탈출 암호로 종단 연결.
D-005
트레일러 영상 프롬프트 완성 — Hailuo AI 기준 6초 × 5씬. LIVE ACTION ONLY · East Asian teenage girls · 35mm film grain.
D-006
ARG 홈페이지 소스코드 완성 — ?ref=find_me(규리 메모) · ?code=EDEN-0047(엔딩 화면) · 콘솔 단서 · HTML 주석 단서.
D-007
불일치 해소 원리 2겹 적용 — 1겹: 수인(조력자→빌런). 2겹: 아란(위협→구원자). Chapter 11 이론 직접 구현.
D-008
Chapter 12 문제 유형 7가지 매핑 — 수색/관찰·개수세기·패턴규칙·카이사르암호·색상치환·논리추론·그림퍼즐 전부 에덴 스토리 적용.
D-009
기술 스택 확정 — HTML 단일 파일 · Vanilla JS · CSS Variables · Google Fonts. localStorage 금지 → JS 변수 상태 관리.
D-010
Flow 이론 난이도 곡선 확정 — Stage 1(관찰·개수 1~2) → Stage 2(패턴·암호 2~3) → Stage 3(논리·복합 4~5).
D-011
엔딩 7개 확정 — 탈출·진실·완전해방·도덕패배·생존·완전해방C·게임오버. 신뢰도 × HQ완료 × 선택지로 분기.
