# 에덴 기숙학원의 비밀 — 프로젝트 INSTRUCTION

## 프로젝트 개요
방탈출 웹/PC 게임 개발. 브라우저에서 바로 실행 가능한 HTML 단일 파일.
ARG(대체현실게임) + 방탈출 + 디지털 스토리텔링 이론을 통합한 게임.

---

## 핵심 설정 (변경 불가)

### 세계관
- 에덴 기숙학원: 영재 양성 명목의 생체실험 기관
- TINAG 원칙 (This Is Not A Game): 현실처럼 위장
- ARG Rabbit Hole → 방탈출 → 엔딩 공유의 순환 구조

### 캐릭터
| 이름 | 역할 | 외형 | 특징 |
|------|------|------|------|
| 권서연 | 주인공(플레이어) | 긴 머리, 청순한 인상 | 실행력, 높은 자존감 |
| 황수인 | 빌런/간부 | 긴 머리 + 비녀, 차가운 인상 | 완벽한 모범생 연기 |
| 김아란 | 진짜 구원자 | 칼단발, 센 인상 | 거칠지만 정의로움 |
| 김규리 | 구조 대상 | 귀여운 인상 | 꼼꼼한 기록, 손재주 |

### 교복
- 네이비색 블레이저 + 체크무늬 넥타이 (네이비/다크레드)

### 시나리오
- **A — 에덴의 가면**: 심리 스릴러 × 바이오호러 (11개 퀘스트)
- **B — 무균실의 기억**: SF 디스토피아 × 탈출 미스터리 (7개 퀘스트)
- **C — 선택받은 자들**: 도덕 공포 × 사회 비판 스릴러 (6개 퀘스트)

## 기술 제약

### 출력 형식
- HTML 단일 파일 (.html)
- 외부 CDN 허용 (Google Fonts, 이미지 없이 CSS/JS만)
- localStorage 사용 금지 (Claude.ai 아티팩트 제약)
- 모든 게임 상태는 JS 변수(메모리)로 관리

### 디자인 시스템
```
배경:       #08080f (최다크) / #0d0d1a / #111128
주색:       #534AB7 (퍼플)
보조색:     #9988FF (퍼플 라이트)
틸:         #1aaa80
앰버:       #c8841a
코랄:       #cc4422
골드:       #c8a84a
텍스트:     #e8e6f5 / #9896b8 (보조)
폰트:       Noto Serif KR (제목) + Noto Sans KR (본문)
모노:       Share Tech Mono (코드/암호)
```

## 문제 유형 설계 원칙 (Chapter 12 적용)

| 유형 | 에덴 적용 방식 | 배치 퀘스트 |
|------|---------------|------------|
| 수색/관찰 | 소품 클릭 인터랙션 | 모든 스테이지 |
| 개수 세기 | 사진 속 특정 물건 카운트 | A-04, C-02 |
| 패턴 규칙 | 기호→숫자 치환표 | A-03, B-01 |
| 카이사르 암호 | 규리 노트 해독 | A-06, B-03 |
| 색상 치환 | 약품 라벨 색상 코드 | A-06 |
| 논리 추론 | 수인 vs 아란 발화 진실 판별 | A-09, C-05 |
| 그림 퍼즐 | 찢어진 쪽지 조각 맞추기 | A-08 |

### Flow 이론 적용 (난이도 곡선)
```
Stage 1: 관찰/개수(난이도 1~2) → 몰입 진입
Stage 2: 패턴/암호(난이도 2~3) → 긴장 상승
Stage 3: 논리/복합(난이도 4~5) → 클라이맥스
```
